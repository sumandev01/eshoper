User Manual - eShopper Website Management

Welcome to the eShopper Content Management Guide. This document explains how you can manage your website's content, specifically the navigation menus, without any coding knowledge.

Managing Menus (Menu Builder)

The Menu Builder allows you to fully control all navigation links on your website. There are 4 distinct menu locations you can manage:

  • Header Top Menu (Top right corner, e.g., FAQs, Help)
  • Main Navigation Menu (The primary navbar with Home, Shop, etc.)
  • Footer Company Menu (First column in the footer)
  • Footer Support Menu (Second column in the footer)

Accessing the Menu Builder

  1. Log in to your Admin Dashboard.
  2. In the left sidebar, click on "Menu Builder".

Selecting a Menu to Edit

  1. On the left side of the Menu Builder page, you will see a dropdown labeled "Select Menu".
  2. Select the menu you want to modify (e.g., "Main Navigation Menu").
  3. Click the "Select" button.
  4. The right side of the screen will now display all the current links inside that menu.

Adding a New Menu Item

Once a menu is selected, you can add new links using the form on the left. There are 4 types of links you can add:

1. System Pages (Recommended for default links)

Use this for standard website pages that are built into the system.

  • Click the "System Pages" tab.
  • Select the page from the dropdown (Home, Our Shop, About Us, Contact Us, FAQs, Order Tracking).
  • Optional: Type a custom Title (if left blank, it will use the default name).
  • Click "Add to Menu".

2. Dynamic Pages

Use this to link to custom pages you created via the "Pages" menu (e.g., Privacy Policy).

  • Click the "Dynamic Pages" tab.
  • Select your created page from the dropdown.
  • Type the Link Title (e.g., "Privacy Policy").
  • Click "Add to Menu".

3. Categories

Use this to add a direct link to a product category in your menu.

  • Click the "Categories" tab.
  • Select the category from the dropdown.
  • Type the Link Title (e.g., "Men's Clothing").
  • Click "Add to Menu".

4. Custom Link

Use this to link to external websites (like your Facebook page) or specific custom URLs.

  • Click the "Custom Link" tab.
  • Paste the full URL (e.g., https://facebook.com/yourpage).
  • Type the Link Title (e.g., "Facebook").
  • Click "Add to Menu".
Note: You do not need to type URLs for system pages, dynamic pages, or categories. The system will automatically generate the correct links for you!

Removing a Menu Item

  1. Look at the right side of the screen where your menu items are listed.
  2. Click the red "Delete" (Trash can) icon next to the link you want to remove.
  3. The link will instantly be removed from your website.
