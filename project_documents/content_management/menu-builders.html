Content Management Guide

A step-by-step guide for non-developers on how to edit, add, and delete website content from the Admin Panel.

1. Introduction

Welcome to the E-Shopper Content Management Guide. You do not need any coding knowledge to use these features. The Admin Panel gives you full control over what appears on your website.

2. Managing Menus (Menu Builder)

The Menu Builder allows you to fully control all navigation links on your website (Header, Footer, etc.).

How to Add a Link to a Menu

  1. Log in to your Admin Dashboard and click on "Menu Builder".
  2. On the left side, select the menu you want to edit from the dropdown at the top of the "Menu Structure" box.
  3. Use the "Add Menu Item" form on the right side to add a link. Choose from one of the four tabs:
    • System Pages: For standard pages like Home, Products, or Contact Us.
    • Dynamic Pages: For custom pages you created (like Privacy Policy).
    • Categories: For linking directly to product categories.
    • Custom Link: For external links like Facebook.
  4. Click "Add to Menu". The link will instantly appear on your website!

How to Edit or Delete a Link

  1. Select the menu you want to edit from the left dropdown.
  2. Look at the left side of the screen where all active links are listed.
  3. Click the blue Pencil (Edit) icon to rename the link.
  4. Click the red Trash Can (Delete) icon to remove the link permanently.

3. Managing Pages

You can create unlimited custom pages like "Terms & Conditions" or "Return Policy" using the Pages module.

How to Create a New Page

  1. Go to the Admin Dashboard and click "Pages".
  2. Click the "Add New Page" button.
  3. Enter the Page Title.
  4. Use the rich text editor to write your Page Content. You can format text, add lists, and make it look beautiful.
  5. Make sure Active Status is checked.
  6. Click "Save Page".
  7. Note: To show this page on the website, go to the Menu Builder and add it!

How to Edit or Delete a Page

  1. Go to "Pages" in the Admin Dashboard.
  2. You will see a table of all your created pages.
  3. Click the blue Edit button to change the text/content.
  4. Click the red Delete button to permanently remove the page.
