Code Documentation

This document provides a guide on how to use the custom Blade components located in resources/views/components.

1. Input (<x-input />)

A versatile text input component that handles labels, validation errors, old input values, and optional Bootstrap input groups.

Properties

PropertyTypeDescription
name RequiredStringThe name attribute of the input field.
labelStringThe text label for the input.
typeStringHTML input type (e.g., text, number, email). Default: text.
valueStringDefault value of the input.
requiredBooleanAdds HTML5 required validation and a red asterisk to the label.
inputGroupBooleanWraps the input in a Bootstrap input-group.
inputGroupTextStringThe text/icon to show in the input group addon.

Usage Example

<x-input 
    label="Product Name" 
    name="title" 
    type="text" 
    placeholder="Enter product name" 
    :required="true" 
    :value="$product->title" 
/>

2. Textarea (<x-textarea />)

A smart textarea component that functions as either a standard HTML textarea (with live character counting) or a fully featured Rich Text Editor (Quill.js).

Properties

PropertyTypeDescription
name RequiredStringThe name attribute of the textarea.
labelStringThe text label for the textarea.
editorBooleanIf true, initializes Quill.js Rich Text Editor. Default: false.
valueStringDefault text/HTML value.
maxlengthIntegerMaximum allowed characters.
rowsIntegerHeight of the editor/textarea in rows. Default: 5.

Usage Example

<!-- Standard Textarea -->
<x-textarea label="Short Description" name="short_desc" />

<!-- Rich Text Editor (Quill) -->
<x-textarea label="Page Content" name="content" :editor="true" />

3. Select (<x-select />)

A dropdown select component that loops through an array or collection of options automatically.

Properties

PropertyTypeDescription
name RequiredStringThe name attribute of the select field.
labelStringThe text label for the select dropdown.
optionsArrayAssociative array for dropdown options.
valueStringThe currently selected option value.

Usage Example

<x-select 
    label="Category" 
    name="category_id" 
    :options="[1 => 'Electronics', 2 => 'Fashion']" 
    value="2" 
/>

4. Color Picker (<x-color-picker />)

A custom UI component for selecting colors.

Usage Example

<x-color-picker 
    label="Product Theme Color" 
    name="theme_color" 
    value="#ff0000" 
/>

5. Media Gallery & Thumbnail (<x-media-gallery /> & <x-media-thumbnail />)

Custom components designed to integrate with the system's media manager. Used for uploading or selecting single thumbnails or multiple gallery images.

Properties

PropertyTypeDescription
labelStringLabel for the upload button/area.
target_idStringThe DOM ID to link with the media manager script.

Usage Example

<!-- Single Image -->
<x-media-thumbnail label="Upload Product Image" target_id="main_thumb" />

<!-- Multiple Images -->
<x-media-gallery label="Product Gallery" target_id="gallerys" />
